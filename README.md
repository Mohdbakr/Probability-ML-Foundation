# Probability & ML Foundations

An open educational website covering practical data science and machine learning topics — from applied statistics and probability to business analytics, modelling, and the mathematical foundations that underpin modern ML.

Topics span (but are not limited to): statistical thinking for business problems, Bayesian reasoning, probability distributions, hypothesis testing, regression, classification, dimensionality reduction, and key mathematical structures in ML such as linear algebra, matrix calculus, and information theory.

## Who it's for

Professionals in analytics, finance, audit, or operations who want to build rigorous quantitative intuition — not just run tools, but understand what they're doing and why.

## Structure

Content is published as lesson pages in curriculum order. The site is bilingual — English (`/`) and Arabic (`/ar/`).

## Tech stack

Built with [Quarto](https://quarto.org), deployed to GitHub Pages via GitHub Actions.
